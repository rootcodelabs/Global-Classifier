# ID-tarkvara toetatud operatsioonisüsteemid - ID.ee

ID-tarkvara on võimalik paigaldada ja uuendada Windowsi, macOS ja Ubuntu ID-tarkvara poolt toetatud operatsioonisüsteemidel. ID-tarkvara ei toeta vanu operatsioonisüsteeme (alates 2020. aastast Windows 7 ja 2022. aasta kevadest Windows 8.1, Windows Server 2012 R2 ja Windows 32-bitiseid operatsioonisüsteeme jne), sest tootja ei paku neile enam uuendusi ja nendele ei arendata ka hädavajalikke tarkvarakomponente, mis mõjutavad ID-tarkvara toimimist nendes süsteemides. See tähendab, et isegi kui ID-tarkvara saaks nendel operatsioonisüsteemidel uuendada, siis RIA-st sõltumatutel põhjustel ID-kaart nendes operatsioonisüsteemides ikkagi ei töötaks. Lisaks kätkeb operatsioonisüsteemide, millele tootja enam hädavajalikke uuendusi ei paku, kasutamine suurt turvariski.
Soovitame ID-tarkvara kasutamiseks uuendada arvuti operatsioonisüsteemi. ID-tarkvara poolt toetatud operatsioonisüsteemide nimekiri on leitav allpoolt.
Kui operatsioonisüsteemi uuendamine pole kohe võimalik, saab digiallkirjastamiseks kasutada mobiili-ID (https://www.id.ee/artikkel/mobiil-id-taotlemine-ja-aktiveerimine/) või nutiseadmetele loodud rakendust nimega RIA DigiDoc (https://www.id.ee/artikkel/ria-digidoc-mobiilirakendus/) .
Kõige värskema tarkvaraversiooni saab alati alla laadida id.ee (https://www.id.ee/artikkel/paigalda-id-tarkvara/) veebilehelt, ehkki ID-tarkvaral on ka muid ametlikke levikanaleid.
ID-kaardi kasutamiseks oma arvutis vajad ID-tarkvara. Siit lehelt leiad alati värskeima tarkvaraversiooni alla laadimise. Salvesta tarkvarafail oma arvutisse ja käivita installeerimine, tehes faili nimel hiirega topeltklõpsu.

ID.ee veebis olev ID-tarkvara paigaldamise leht:
Meie ID-tarkvara paigaldamise (https://www.id.ee/artikkel/paigalda-id-tarkvara/) lehe allalaadimise lingid on alati värskendatud ning pakuvad automaatselt selle tarkvarapaketi, millise operatsioonisüsteemiga arvutiga antud lehele minnakse.
Microsoft Windows Store:
Microsoft Store (https://www.microsoft.com/et-ee/store/apps/windows) kaudu saad alla laadida ID-tarkvara Windows (https://www.microsoft.com/et-ee/p/digidoc4-klient/9pfpfk4dj1s6) jaoks, näiteks:

- DigiDoc4, mis on mõeldud ID-kaardi kasutamiseks arvutis.
Apple'i App Store:
Apple’i App Store (https://apps.apple.com/) kaudu saad alla laadida ID-tarkvara macOS (https://apps.apple.com/us/developer/ria/id556524921) jaoks, näiteks:

- DigiDoc4, mis on mõeldud ID-kaardi kasutamiseks arvutis.
- Web eID komponent, mis on mõeldud ID-kaardi kasutamiseks veebibrauserites.
- RIA DigiDoc mobiilirakendus iOS nutiseadmetele.

Windows Update kaudu paigaldatakse arvutisse automaatselt ID-tarkvara osaks olev minidraiveri komponent, mis võimaldab ID-kaardiga elektroonilisi toiminguid teha.

Digiallkirjastatud failide avamiseks ja digiallkirjastamiseks olemas tarkvara ka Android ja iOS seadmetele:
- Android: Google Play'st (https://play.google.com/store/apps/details?id=ee.ria. DigiDoc)
- iOS: App Store'st (https://itunes.apple.com/ee/app/ria-digidoc/id1216104448?mt=8)
