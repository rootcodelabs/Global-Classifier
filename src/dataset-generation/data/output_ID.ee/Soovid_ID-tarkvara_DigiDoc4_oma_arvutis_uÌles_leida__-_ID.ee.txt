# Soovid ID-tarkvara DigiDoc4 oma arvutis üles leida? - ID.ee

Kas oled oma arvutisse ID-tarkvara paigaldanud ega suuda nüüd ID-kaardi kasutamiseks DigiDoc4 rakendust leida?
(#digidoc4-kaivitamine-win)
Windowsi operatsioonisüsteemides luuakse ID-kaardi tarkvara paigalduse käigus otsetee ehk sinu arvuti töölauale ( ) juhul, kui sa paigalduse käigus ei määranud teisiti.
DigiDoc4 avamiseks tee arvuti töölaual oleval DigiDoc4 ikoonil hiirega topeltklõps.
Kui sul ei ole töölaual ikooni, millel klõpsata, siis üldjuhul on olemas otsetee ka START-menüüs.
kasutajad leiavad DigiDoc4 Start menüüst järgmiselt: D tähe all on rakenduse nimega DigiDoc4.
kasutajad leiavad DigiDoc4 ikooni Start menüüst, kui vajutavad lisaks veel valikule >, siis leiavad D tähe alt . Avamiseks tee valik hiirega.
(#digidoc4-kaivitamine-macos)
tarkvaras paigaldatakse otsetee ikoon süsteemsesse kataloogi või sinu töölauale ( ). Avamiseks klõpsa hiirega DigiDoc4 ikoonil.
Lisaks võib kasutada ülal paremal nurgas otsingut märksõnaga .
(#digidoc4-kaivitamine-linuxis)
DigiDoc4 rakenduse otsetee ikooni asukoht võib erineda sõltuvalt kasutusel olevast Linux versioonist. Parim moodus on kasutada süsteemset otsingut, mis on olemas kõikidel Linuxi versioonidel. Otsingul kasuta märksõna või .

Üleval vasakus nurgas on süsteemse otsingu otsetee ikoon ( ), klõpsa sellel ja kirjuta otsisõnaks  ID. DigiDoc4 avamiseks klõpsa hiirega ikoonil.
Kas oled oma arvutisse ID-tarkvara paigaldanud ega suuda nüüd ID-kaardi kasutamiseks DigiDoc4 programmi leida? Juhised Windows, macOS ja Linux operatsioonisüsteemis DigiDoc4 käivitamiseks.
