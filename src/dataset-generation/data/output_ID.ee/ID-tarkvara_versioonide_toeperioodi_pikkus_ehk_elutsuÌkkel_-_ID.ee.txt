# ID-tarkvara versioonide toeperioodi pikkus ehk elutsükkel - ID.ee

ID-tarkvara on ja seepärast on väga oluline kasutada alati ainult kõige uuemat tarkvaraversiooni. ID-tarkvaral on olemas automaatse uuendamise süsteem – kindlasti tuleks esimesel võimalusel paigaldada kõik uuendused või parendused, millest ID-tarkvara sulle teada annab!
Tarkvaraversiooni toeperioodi lõpp tähendab, et:

- ID kasutajatugi ei uuenda enam versioonikohast teavet teadmistebaasis
- Antud versioonis ei testita enam viidatud vigade olemasolu
- Tehnilise lahenduse muudatusettepanekuid enam vastu ei võeta
- Funktsionaalsuste muutmise/parendamise taotlusi vastu ei võeta
- Operatsioonisüsteemi ja ID-tarkvara koostoimeprobleeme ei lahendata
Kasutades tarkvara, mille toeperiood on lõppenud, võtab kasutaja endale kogu vastutuse vananenud tarkvara kasutamisest tingitud probleemide eest.
