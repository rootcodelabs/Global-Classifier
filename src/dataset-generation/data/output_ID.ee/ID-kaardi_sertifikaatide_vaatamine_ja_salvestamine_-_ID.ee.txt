# ID-kaardi sertifikaatide vaatamine ja salvestamine - ID.ee

Sul on alati võimalus vaadata ning salvestada oma ID-kaardi sertifikaate. Kõige mugavam on seda teha DigiDoc4 abil.
Ühenda kaardilugeja arvutiga, sisesta sellesse oma ID-kaart, ava DigiDoc4 ning oota, kuni sinu ID-kaardi andmed on sisse loetud. Edasised juhised sõltuvad natuke sellest, millise operatsioonisüsteemiga arvutit kasutad. Vali allolevast valikust endale sobivaim:

Vali sealt kas või ja klõpsa tekstilingil "Vaata sertifikaadi detaile".
Sertifikaadi salvestamiseks vali avanenud aknast „Üksikasjad“ ( ) ja vali „Kopeeri faili“ ( ) ning klõpsa avanenud aknast nupul „Edasi“ ( ).

Siis vali kaust, kuhu soovid sertifikaadi salvestada, ning vajuta veelkord „Edasi“ ( ):

Kui sertifikaat salvestatakse edukalt, kuvatakse selle kohta teade (Eksport oli edukas):

Seejärel vali või alt tekstilink "Vaata sertifikaadi detaile".

Avaneb uus aken koos sertifikaadi detailidega. Juhul, kui soovid sertifikaadi salvestada, vajuta ekraani ülemises paremas nurgas oleval „share“ nupul (kast ja nool üles) ning vali tekkinud menüüst programm „Notes“.
Sulle avatakse nüüd Notes programmi salvestusaken – kui soovid sertifikaadi eraldi failina salvestada, vajuta „Save“ nupul.

Sertifikaadi kasutamiseks ava arvutis programm "Notes" ning lohista nähtavale ilmunud sertifikaadifail endale sobivasse kataloogi, näiteks Desktopile. Samuti saad sertifikaadi topeltklõpsuga Notes’is avada.
Seejärel vali vali või alt "Vaata sertifikaadi detaile"

Sertifikaadi detailvaates on sul võimalik sertifikaadi info salvestada – ekraani allosas asuval „Salvesta“ nupul klikates.
Juhised, kuidas vaadata ja salvestada oma ID-kaardi sertifikaate.
