# Mobiil-ID: digitaalne isikutunnistus nutitelefonis - ID.ee

Mobiil-ID on SIM-kaardi põhine digitaalne isikutunnistus, mis on koos nutiseadmega sinuga alati kaasas.
Täpselt nagu ID-kaart (digi-ID, elamisloakaart jne) on ka mobiil-ID riikliku garantiiga isikutunnistus, mille sertifikaate väljastab Politsei- ja Piirivalveamet. Mobiil-ID abil saad teha kõiki neid toiminguid, milleks muidu kasutad elektrooniliselt ID-kaarti: siseneda e-teenustesse, näiteks panka; kinnitada teenusemuudatusi ja pangaülekandeid; allkirjastada dokumente; siseneda riiklikesse registritesse; hääletada valimistel jpm.

- Taotlemine (https://www.id.ee/artikkel/mobiil-id-taotlemine/)
- Kasutamine (https://www.id.ee/artikkel/mobiil-id-kasutamine/)
- PIN-koodid (https://www.id.ee/artikkel/mobiil-id-pin-koodid/)
- Tutvustus (https://www.id.ee/rubriik/tutvustus-mobiil-id/)
