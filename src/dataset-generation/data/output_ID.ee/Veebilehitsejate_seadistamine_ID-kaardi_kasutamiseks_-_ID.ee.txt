# Veebilehitsejate seadistamine ID-kaardi kasutamiseks - ID.ee

Selleks, et saaksid ID-kaarti veebilehitsejates kasutada – näiteks internetipanka, Eesti.ee keskkonda ja muudesse e-teenustesse sisenemiseks, tegevuste ja tehingute kinnitamiseks, digiallkirjastamiseks jne. – on vajalik, et veebilehitseja lubaks ID-kaardi sertifikaatide kasutamist.
Mobiilsete seadmete (telefon, tahvelarvuti jne.) veebilehitsejas ei ole Web eID laiendust. Web eID laiendus on ainult arvuti veebilehitsejas.
Kui sul on tekkinud tõrkeid ID-kaardi kasutamisel veebis, kontrolli üle oma lehitseja seaded. Meie samm-sammuliste juhiste abil käib veebilehitsejate seadistamine kiirelt ja lihtsalt!

Kui laienduse paigaldamine ei ole mingil põhjusel võimalik, toimige järgmiselt:
- Alates Ubuntu 21.10 versioonist on vaikimisi paigaldatud Firefox SNAP-na, milles ei saa ID-kaarti kasutada veebis autentimiseks ja allkirjastamiseks. Olukorra lahendamiseks Firefoxi veebilehitsejas ID-kaardiga veebis autentimiseks ja allkirjastamiseks tuleks paigaldada Firefoxi tavaversioon. (Firefoxi SNAP-i eemaldamine eemaldab KÕIGI kasutajate Firefox profiilid. Soovitame enne Firefox SNAP eemaldamist Mozilla Firefox profiilid varundada). Selleks leiate juhised siit (https://www.id.ee/artikkel/ubuntu-id-tarkvara-paigaldamine-uuendamine-ja-eemaldamine/#mozilla-firefox-snap) .
- Kontrollige Mozilla Firefoxi värskendusi. Kui neid on, paigaldage need ning taaskäivitage veebilehitseja kaks korda.
- Kui teil on viirusetõrjetarkvara olemas, konfigureerige see vastavalt (https://www.id.ee/rubriik/turvatarkvara/) või eemaldage see ajutiselt.
