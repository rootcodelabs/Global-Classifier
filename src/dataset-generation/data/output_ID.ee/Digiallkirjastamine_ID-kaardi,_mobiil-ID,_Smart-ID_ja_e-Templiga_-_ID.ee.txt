# Digiallkirjastamine ID-kaardi, mobiil-ID, Smart-ID ja e-Templiga - ID.ee

ID-kaardi, mobiil-ID, Smart-ID ja e-Templiga digiallkirjastamise realiseerimiseks enda e-teenuses on erinevaid võimalusi. Selleks, et leida endale sobivam lahendus, soovitame tutvuda kõikide allpool olevate võimalustega.
Valitsussektori asutustel on võimalik ID-kaardi, mobiil-ID, Smart-ID ja e-Templiga digiallkirjastamiseks enda e-teenuses liituda Riigi allkirjastamisteenusega (https://www.ria.ee/riigi-infosusteem/elektrooniline-identiteet-ja-usaldusteenused/digiallkirja-serverteenused) . Soovi korral on võimalik kõikidel huvilistel kasutada ka Riigi allkirjastamisteenuse aluseks olevat tarkvara. (https://github.com/open-eid/SiGa)
Digiallkirjade kontrollimiseks on võimalik oma e-teenus liidestada valideerimisteenus SiVa’ga (https://www.ria.ee/riigi-infosusteem/elektrooniline-identiteet-ja-usaldusteenused/digiallkirja-serverteenused) , mille abil saab lihtsasti kontrollida digiallkirjastatud dokumente.
DigiDoc teekide abil on võimalik integreerida digiallkirjade loomist ja valideerimist enda süsteemidesse ja rakendustesse:

- Java teek: DigiDoc4j (https://www.id.ee/artikkel/digidoc-teegid-java-teek-digidoc4j/)
- Multiplatvormne C++ teek: Libdigidocpp (https://www.id.ee/artikkel/digidoc-teegid-c-teek-libdigidocpp/)
ID-kaardiga digiallkirjastamise realiseerimiseks enda e-teenuses:

- Digiallkirjastamine ID-kaardiga (https://www.id.ee/artikkel/digiallkirjastamine-id-kaardiga/)
Mobiil-IDga digiallkirjastamise realiseerimiseks enda e-teenuses:
- Isikutuvastus ja digiallkirjastamine mobiil-ID’ga (https://www.id.ee/artikkel/kuidas-mobiil-id-oma-infosusteemiga-siduda-3/)
Smart-IDga digiallkirjastamise realiseerimiseks enda e-teenuses:

- https://github.com/SK-EID/smart-id-documentation (https://github.com/SK-EID/smart-id-documentation)
Muud erasektori poolt pakutavad isikutuvastuse ja digiallkirjastamise lahendused:

- Dokobiti arendajasõbralikud autentimise ja digiallkirjastamise lahendused (https://www.id.ee/artikkel/dokobiti-arendajasobralikud-autentimise-ja-digiallkirjastamise-lahendused/)
- eID Easy – lahendus eID arendajatele (https://www.id.ee/artikkel/eid-easy-lahendus/)
- Undersign.js - JavaScripti teek digiallkirjade loomiseks (https://www.id.ee/artikkel/undersign-js-javascripti-teek-digiallkirjade-loomiseks/)

- Teenuste testimine: üldinfo (https://www.id.ee/artikkel/teenuste-testimine-uldinfo/)
Selleks, et oleks lihtsam lisada veebirakendustesse allkirjastamise tuge, soovitame kasutada JavaScript teeki, mille API on ühtne ja mis võimaldab erinevaid pluginaid (operatsioonisüsteemist ja brauserist sõltumata) ühte moodi kasutada.
