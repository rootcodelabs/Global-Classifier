# ID-tarkvara kõige uuema versiooni info - ID.ee

Kõige värskema ID-tarkvara paketi leiad alati lehelt. Meie veebileht kontrollib automaatselt, millist operatsioonisüsteemi sa kasutad, ning pakub sulle alla laadimiseks just sinu arvuti jaoks sobivaima(d) tarkvarapaki(d). Sul pole vaja teha muud, kui tarkvara alla laadida ning arvutis paigaldamine käivitada!
(#kuidas-tarkvara)
ID-tarkvara, nagu kõiki tarkvarasid, eristatakse versiooninumbri järgi. Mida suurem on number, seda uuem ehk värskem antud tarkvara on. Kui sul on vaja valida, millist versiooni alla laadima hakkad, eelista alati kõige uuemaid tarkvaralahendusi! Nii võid olla kindel, et oled võimalikult paljude turvariskide eest kaitstud, sinu nutiseade/arvuti on töökindel ning sinu käsutuses on kõige viimasem, kasutajamugavam ja kiirem versioon töövahenditest.
Tegelikult koosneb ID-tarkvara (ehk eID tarkvara) paljudest erinevatest komponentidest, mida me sinu elu lihtsustamiseks ühtse „pakina“ edastame. Oma versiooninumber on olemas ka igal komponendil eraldi: lisaks uudsusele võib versiooninumber eristada ka operatsioonisüsteeme, mille jaoks antud komponent mõeldud on.
Alljärgnevad tabelid annavad ülevaate kõige uuema ID-tarkvara komponentide versioonidest.
NB!  Kõik vajalikud tarkvara komponendid paigaldatakse arvutisse ID-tarkvara paigalduspaketi poolt, mille saab ID-tarkvara paigaldamise lehelt. Erandiks on macOS kasutajate jaoks App Store kaudu levitatav Web eID ja DigiDoc4 rakendus.
