# Teavita meid probleemist - ID.ee

Kõikidele suurematele ID-kaardi ja mobiil-ID elektroonilist kasutamist puudutavatele küsimustele leiad vastused id.ee veebist: kasuta julgesti meie otsingut! Kui sinu mure jääb aga lahendamata, võta ühendust ID-abikeskusega siinse vormi kaudu. Mida täpsemalt kirjeldad oma probleemi ja selle olukorra tekkimist, seda personaalsemat abi suudame sulle pakkuda. Kõik vastused saadame sulle kirjalikult ja lihtsalt järgitavate juhistena!
