# Mis see on? - Digi-ID taotlemine - Politsei- ja Piirivalveamet

Mis see on?
Digitaalne isikutunnistus ehk digi-ID on digitaalne dokument, millega saab elektroonilises keskkonnas oma isikut tuvastada ja anda digitaalset allkirja.

Digi-ID näidates ei saa enda isikut tõendada.

Digi-ID saad taotleda juhul, kui oled:

- Eesti kodanik, kellel on kehtiv ID-kaart või kui taotled digi-ID koos ID-kaardiga
- välismaalane, kellel on kehtiv elamisloakaart/ID-kaart või kui taotled digi-ID koos elamisloakaardiga

Digi-ID kehtivusaeg on 5 aastat.

Sertifikaatide kasutustingimused. (https://www.id.ee/kasutustingimused)
ID kasutajatugi www.id.ee (https://www.id.ee/) Sertifikaatide peatamine 677 3377
